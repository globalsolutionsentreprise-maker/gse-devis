🔗 Connexion Google Drive

Pour sauvegarder vos devis dans le Drive de globalsolutionsentreprise@gmail.com, suivez ces étapes :

  1. Allez sur console.cloud.google.com
  2. Créez un projet (ex : "GSE-Devis")
  3. Activez l'API Google Drive
  4. Créez des identifiants → Clé API (et restreignez à l'API Drive)
  5. Créez aussi un ID client OAuth 2.0 → Application web
  6. Collez l'ID client OAuth ci-dessous et cliquez "Connecter"

💡 Alternative plus simple : utilisez un dossier public partagé → collez l'ID du dossier Drive et l'API Key ci-dessous.

Comment ça marche ?

Chaque devis enregistré est sauvegardé en 2 endroits :
① Localement dans votre navigateur (toujours disponible hors ligne)
② Sur Google Drive en tant que fichier JSON dans le dossier configuré

Vous pouvez aussi imprimer en PDF depuis le bouton "Imprimer / PDF" et l'envoyer manuellement par email.
